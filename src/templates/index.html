How To Use

  1. Download and install OpenVPN Connect from this link https://openvpn.net/client/
  2. Download the OpenVPN config from the button above
  3. Extract the downloaded zip file
  4. Launch OpenVPN Connect and chose import file
  5. Drag and drop the extracted zip files to the OpenVPN Connect UI
  6. Now you can chose which VPN gateway you want to connect to
